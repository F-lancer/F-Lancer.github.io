计算机基础

最近也在看TS，顺便看一下后端语言，康康和JS的区别，发现较大的web项目都是用Java写的，就来了解一下。了解了强类型才能深刻了解弱类型，了解重载，才囊知道非重载，一点一点连成片…..

二进制

二进制：逢二进一，每一个0或1叫bit（位）。

十进制转二进制：除以2求余。

字节

字节是数据存储单元的最小单位，八位（bit）一个字节（Byte）

1 Byte = 8 bit | 1 KB = 1024 Byte | 1 MB = 1024 KB | 1GB = 1024 MB | 1TB = 1024 GB | 1PB = 1024 TB | 1EB = 1024 PB | 1ZB = 1024 EB

CMD

windows 命令提示符。

  1. 快捷键：WIN +R。
  2. 切换盘符：cd/Tab。
  3. ..返回上级文件夹。
  4. dir 列表。
  5. cls清屏。
  6. exit退出。

开发环境

JVM

Java Virtual Machine：java虚拟机。

Java 运行在 JVM 上，JVM 的不同版本运行在各个系统上，即由各个系统的不同版本的虚拟机来实现跨平台。

JDK

开发工具包

配置环境 变量

编译运行

java三步骤：编写、编译、（JVM）运行。

javac.exe：编译器。 源代码经过javac.exe编译后生成.class文件。

java.exe：解释器。运行时不带 后缀名。

注意：文件名要和类名保持一致。

注释

helloworld

关键字

特征：全小写、编辑器特殊颜色。

标识符

定义：自定义的内容就是标识符。

命名规则：

  1. 字母（区分大小写）、数字、$、_。
  2. 不能以数字开头。
  3. 不能是关键字。

命名规范：

  1. 类名规范：首字母大写（大驼峰）。
  2. 变量及方法名规范：首字母小写（小驼峰）。

常量

定义：在 java 程序运行的期间固定不变的数据。

分类：

  1. 字符串常量：双引号引起来的部分。
  2. 整数常量。
  3. 浮点数常量。
  4. 字符常量：单引号引起来的单个（有且仅有一个）字符。
  5. 布尔常量。
  6. 空常量。（不能直接打印输出，报错）

数据类型

基本数据类型

四类八种。

整数型：byte（1Byte） short（2Byte） int（4Byte） long（8Byte）

浮点型：float（4Byte） double（8Byte 更精确）

字符型：char（2Byte）

布尔型：boolen （1Byte）

  1. 字符串不是基本类型。
  2. 浮点数是近似值。
  3. 数据范围与字节数不一定相关
  4. 浮点数默认类型是double，使用float类型要加一个 F 后缀。
  5. 整数默认是int类型，使用long类型，加 L 后缀。

引用数据类型

类、数组、接口。

数据类型转换

自动类型转换

特点：

  1. 代码不需要处理，自动完成。

  2. 需要符合数据范围从小到大的规则（和字节数不一定相关）。

    int —>long —> float—>double

强制类型转换

特点：

  1. 无法自动完成。

  2. 格式：范围小的类型 变量名 = （范围小的类型）大范围数据;

    int num = (int) 100L

  3. 逻辑上 范围大 转小会发生数据溢出。

编码表

Ascll编码表：0~127

American Standard Code for Information Interchange：美国信息交换标准代码。

Unicode码表：万国码。从128开始添加了其他字符。

打印字符：’a’+ 0 可以查看对应码。

‘0’—‘48’ ‘A’—‘65’ ‘a’—‘97’

变量

定义：内容可以发生改变的量。

注意：

  1. 变量名不能重复。
  2. FL后缀不能省。
  3. 数值不能超过范围。
  4. 先声明后使用。
  5. 不能超出作用域{}。

运算符

四则运算

加+减-乘*除/余%：

  1. 整数的除/，只看商，不看余。

  2. 先转换后计算。

    int + double：double + double = double

  3. 加号+，char类型先转换，后相加，string类型（首字母大写，非关键字）拼接。

  4. 任何类型和字符串连接都转换为字符串。

自加（++）自减（–）

  1. 运算符在前，前加加，先加后使用。否则相反。
  2. 常量不能参与运算。30++。

赋值运算符：+=、-=、*=、/=、%=

  1. 同样常量不能参与运算。
  2. 隐含强制类型转换。

比较运算符，结果为布尔值。

  1. 不能连写，1<x<3 报错，1 < x && x < 3。

逻辑运算符：根据短路原理理解。

三元运算符：指需要三个数据才能进行操作的运算符。

  1. 三元运算符的结果必须被赋值/被使用。

方法

定义

  1. 先后顺序无所谓。
  2. 方法的定义在类中，不能在方法中定义方法产生嵌套包含的关系。
  3. 返回值类型为 void 无返回值不能进行单独调用和赋值调用。

重载

定义：多个方法的名称一样，但是参数类型不一样。

在 JavaScript 中是没有重载的。

java 会自动去寻找参数相同的方法执行，而在 javaScript 中，名称相同的方法，后者会覆盖前者的功能。

注意：

涉及因素：参数的个数、参数类型不同、参数多类型顺序不同。

不涉及的因素：参数名称、方法的返回值类型无关。

JDK9

是最后一个大版本的更新。新特性：

  1. JShell脚本工具。

  2. javac 数值未超过变量类型范围，编译器会自动补上强转，超过则报错。右侧为变量则不进行强转。

流程

顺序结构、判断结构、选择结构、循环结构。

选择结构

switch 语句注意事项：

  1. case语句之后的数值不能重复。

  2. switch 后面的数据类型只能是：（JavaScript可以是任何类型）

    （1）基本数据类型：byte/short/char/int。

    （2）引用数据类型：String字符串、enum枚举。

  3. 顺序随意，break 语句可省略（省略后会直到遇到break，跳出）。

循环结构

for 循环：初始化语句、条件判断、循环体、步进语句。

while 循环：直到逻辑。

do-while循环：先循环，后判断。

区别：do-while至少循环一次，初始化语句for循环只能在语句内部。

break/continue

break：打断循环。

continue：跳出本次循环进入下次循环。

死循环之后不能有语句，会报错。

数组

特点

引用数据类型。

  1. 数组当中的多个数据，类型必须统一。（JavaScript 数组中的每一项可以是任意类型。）
  2. 数组的长度在程序运行期间不可改变。（JavaScript 数组的长度是动态的。）

创建

  1. 动态初始化（指定长度）。

  2. 静态初始化（指定内容）。

使用

直接打印数组名称，得到的是数组对应的，内存地址哈希值。例：[I@75412c2f 数组 int 十六进制数字

动态初始化使用，未赋值拥有默认值。

整数 0 、浮点数0.0、 字符类型 \u0000、 布尔 false、 引用类型 null 。

内存划分

  1. 栈（Stack）：存放的都是方法中的局部变量（方法的参数或内部的变量）。

    作用域：一旦超出作用域，立刻从栈内存中消失。

  2. 堆（Heap）：凡是new出来的东西，都在堆当中。其中的地址值是16进制。且都有默认值。

  3. 方法区（Method Area）：存储 .class 的相关信息，包含方法的信息。（方法运行在栈中）。

  4. 本地方法栈（Native Method Stack）：与操作系统相关。

  5. 寄存器（pc Register）：与 CPU相关。
